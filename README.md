#FNP Sales Data Analysis Project
Project Overview

This project presents a comprehensive sales data analysis of FNP (Ferns N Petals) business performance. The objective is to extract meaningful business insights from sales data and help improve decision-making using data-driven strategies.

The analysis focuses on revenue trends, customer behavior, product performance, and seasonal demand patterns.

Problem Statements

The project answers the following key business questions:

What is the Total Revenue of the business?

How many Total Orders were placed?

What is the Monthly Sales Performance?

Which are the Top Performing Product Categories?

Which occasions generate the most revenue?

Which cities contribute the highest revenue?

What is the Average Customer Spending?
